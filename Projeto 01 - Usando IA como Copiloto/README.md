# Projeto - Usando IA como Copiloto

Como primeiro projeto do curso de Node.js da *Digital Innovation One*, programos que uma Inteligência Artificial, eu escolhi o *Claude*, como copiloto: um agente que auxilia a programação.

Fiz quatro tipos:
 - **Agent**: refere-se a sistemas de inteligência artificial que não apenas geram texto ou imagens, mas agem de forma autônoma ou semi-autônoma para alcançar objetivos específicos. 
 - **Ask**: que são focadas em responder perguntas, gerar textos, analisar documentos e fornecer respostas rápidas.
 - **Plan**: organização, gestão de projetos e elaboração de estratégias. Essas IAs ajudam a criar cronogramas, delegar tarefas, fazer análises SWOT (FOFA) e estruturar projetos do zero a partir de uma ideia.
 - **Study**: que ajudam a resumir textos, criar flashcards, resolver questões e criar mapas mentais automaticamente.
