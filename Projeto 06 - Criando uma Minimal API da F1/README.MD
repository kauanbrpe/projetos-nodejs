# Criando um Minimal Api da Fórmula 1

Dentro da trilha de estudos da Digital Innovation One foi proposto um Minimal Api da Fórmula 1 utilizando o runtime Node.js com a Lingugagem TypeScript.

## 🌐 Tecnologias utilizadas

- JavaScript
- TypeScript (Superset do JavaScript)
- Node.js
- Visual Studio Code
- NPM (Node Package Manager)
- Fastify
